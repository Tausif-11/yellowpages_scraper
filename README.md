# YellowPages Multi-Format Extraction Framework

Production-ready pipeline engineered to scrape Roofing Contractor listings in Los Angeles, CA from YellowPages.

## Architecture
- Bypasses traditional client-side rendering bottlenecks by parsing hidden `application/ld+json` structural elements directly from response text.
- Throttles tracking sequences organically to mimic safe, non-disruptive browsing behavior.
- Compiles unstructured web targets straight into five standardized production outputs: Excel, CSV, JSON, XML, and HTML.
