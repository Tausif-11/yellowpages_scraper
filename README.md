# Industrial-Grade Yellow Pages Lead Generation Scraper

A robust, enterprise-grade Python automation framework engineered to systematically crawl, parse, and extract local B2B business lead datasets from Yellow Pages directories. Explicitly optimized for high-volume lead extraction (e.g., *Roofing Contractors in Los Angeles, CA*), generating deduplicated, commercial-ready cold outreach matrices.

## 🚀 Key Capabilities
- **Multi-Format Export Pipeline:** Simultaneously structures data pipelines and saves cleanly to five distinct structural assets: **Excel (.xlsx), CSV, JSON, XML, and HTML**.
- **Comprehensive Lead Mapping:** Extracts complete business target profiles including Name, Phone, Address, Operating Longevity (Years in Business), Website URLs, and Associated Service Sub-categories.
- **PyPI-Ready Architecture:** Structured completely inside a production `src/` layout matrix for seamless installation and dependency isolation.
- **Enterprise UI Formatting:** Generates custom-styled, reader-optimized corporate tracking spreadsheets utilizing zebra-striping patterns and data type alignment logic.

## 🛠️ Core Tech Stack
- **Data Engine:** Python 3.11+
- **Parsing Architecture:** Beautiful Soup 4 & `lxml`
- **Transport Layer:** `requests` / `tls_client`
- **Data Transformation Matrix:** `pandas`
- **Spreadsheet Design Framework:** `openpyxl`
